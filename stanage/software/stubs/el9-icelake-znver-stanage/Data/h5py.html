HDF5 for Python (h5py) is a general-purpose Python interface to the Hierarchical Data Format library, version 5. HDF5 is a versatile, mature scientific software library designed for the fast, flexible storage of enormous amounts of data.

Note

This is an autogenerated page, more detail including examples may be available for this package. Please see Software on Stanage

After connecting to Stanage (see Establishing a SSH connection), start an interactive session with the following command:

srun --pty bash -i

A version of h5py can then be made available with one of the following:

module load h5py/3.7.0-foss-2022a

module load h5py/3.7.0-foss-2022a

Notes

h5py was installed using Easybuild, build details can be found in $EBROOTH5PY/easybuild with a given module loaded.
