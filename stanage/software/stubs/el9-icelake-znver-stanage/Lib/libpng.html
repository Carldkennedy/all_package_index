Note

This is an autogenerated page, more detail including examples may be available for this package. Please see Software on Stanage

After connecting to Stanage (see Establishing a SSH connection), start an interactive session with the following command:

srun --pty bash -i

A version of libpng can then be made available with one of the following:

module load libpng/1.6.38-GCCcore-12.2.0
module load libpng/1.6.37-GCCcore-11.3.0
module load libpng/1.6.37-GCCcore-11.2.0
module load libpng/1.6.37-GCCcore-10.3.0
module load libpng/1.6.37-GCCcore-10.2.0
module load libpng/1.6.37-GCCcore-9.3.0
module load libpng/1.6.34-GCCcore-9.3.0

module load libpng/1.6.37-GCCcore-11.3.0

libpng was installed using Easybuild, build details can be found in $EBROOTLIBPNG/easybuild with a given module loaded.
